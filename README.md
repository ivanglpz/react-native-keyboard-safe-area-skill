# React Native Keyboard Safe Area Skill

A Codex skill for React Native + Expo screens that need correct `SafeAreaView`, `KeyboardAvoidingView`, `ScrollView`, header, stepper, and sticky-footer behavior across iOS and Android.

## Install

From this repository:

```sh
npx skills add .
```

From GitHub after publishing:

```sh
npx skills add ivanglpz/react-native-keyboard-safe-area-skill --skill keyboard-safe-area
```

## Skill

- `keyboard-safe-area`: production rules for keyboard-safe React Native + Expo screens without hardcoded offsets or device hacks.
